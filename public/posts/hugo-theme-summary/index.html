Hugo Theme Summary

Friday, February 28, 2020

You can change the summary from palin text to rendered text by doing the following:

  1. Go to theme folder/layouts/partials/note-list.html
  2. You will see something like 
    {{- if .Summary -}}<p class="note content">{{- .Summary | plainify | safeHTML -}}{{- if .Truncated -}}<span class="mldr">&mldr;{{- T "more" -}}</span>{{- end -}}</p>{{- end -}}
  3. Replace the word plainify with markdownify
  4. Done.

Furthermore, put <!--more--> ahead of all content, and you will get a simple and clean list with no summary. For more information, you can visit https://yihui.org/en/2017/08/hugo-post-summary/
