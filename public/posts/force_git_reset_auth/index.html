Force Github to Re-Auth

Friday, February 28, 2020

If you forget your Github passphrase you can force the system(e.g. Mac OS) to re-authenticate and ask for the passphrase/token again:

git credential-osxkeychain erase
# Now you enter an input staus
# Simply input the followings:
host=github.com
protocol=https
# Press Enter
git push

Now the git command shall ask for your username and token again. For further information, you can visit here
